speculative work remain executor work kinds; they do not own separate capacity
liveness state. The no-pressure path must keep stable batch membership, allocate
nothing, and create no pressure episode. At a real allocation failure, one scheduler
transaction must either claim work for at least one frontier or plan a typed
yield/fence-delayed release before parking.
